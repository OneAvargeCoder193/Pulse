# Pulse

Pulse is a programming language of time

### Build:
- make LLVM folder
- place bin, include, lib, libexec, share in that folder (you can download them from the github or build it yourself llvm-config isn't necessary)
- run `zig build`

### Usage:
`pulse [file] -o [output] -O [0/1/2/3/s] -t [target] -e [exec/obj/llvm-ir/asm]`
